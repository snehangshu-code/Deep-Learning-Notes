2 RNN Unrolled Through Time
3 Weight Matrices
Wxh
shape: [H × V]

Transforms input vector xₜ (vocab size V) into hidden space (size H). Shared across all timesteps.

Whh
shape: [H × H]

Transforms previous hidden state hₜ₋₁ into current hidden space. This is the recurrent "memory" connection.

Why
shape: [C × H]

Maps final hidden state h₄ to output classes C (positive / negative / neutral).

4 Core Equations

Hidden State Update

hₜ = tanh(Wxh·xₜ + Whh·hₜ₋₁)
At each step, the RNN combines two signals: Wxh·xₜ (what the current word says) and Whh·hₜ₋₁ (what was remembered so far). tanh squashes values to (−1, 1), preventing explosion.

Output Prediction

ŷ = softmax(Why·h₄)
Only at the final timestep (for sentence classification), Why projects the compressed hidden state into class scores. Softmax converts scores into a probability distribution over sentiments.
5 Processing "the food is good" — Step by Step
0

Initialise h₀ = [0, 0, …, 0]

Before any word is seen, the hidden state is a zero vector. No "memory" yet.

1

t=1 · Read "the" → x₁ = [1,0,0,0,0,0]

Compute h₁ = tanh(Wxh·x₁ + Whh·h₀). Since h₀=0, this is just tanh(Wxh·x₁) — extracting the "the" representation. h₁ encodes: "a sentence started with 'the'."

2

t=2 · Read "food" → x₂ = [0,1,0,0,0,0]

Compute h₂ = tanh(Wxh·x₂ + Whh·h₁). Now h₁ is carried forward via Whh. h₂ "knows" the sentence so far: "the food…"

3

t=3 · Read "is" → x₃ = [0,0,1,0,0,0]

Compute h₃ = tanh(Wxh·x₃ + Whh·h₂). The word "is" acts as a transition. h₃ encodes: "the food is…" — the network is primed for a sentiment word.

4

t=4 · Read "good" → x₄ = [0,0,0,1,0,0]

Compute h₄ = tanh(Wxh·x₄ + Whh·h₃). This final hidden state now compresses the entire sentence: "the food is good." Context from all previous words is folded in through the chain of Whh multiplications.

→

Output · ŷ = softmax(Why · h₄)

Project h₄ through Why to get class logits, then softmax. Expected result: high probability on Positive, low on Negative / Neutral — because "good" is the dominant sentiment signal.

KEY INSIGHT

Compare with "the food is bad" → x₄=[0,0,0,0,1,0] would push h₄ toward negative sentiment.
And "the food is not good" → the word not (x₄=[0,0,0,0,0,1]) enters h₄ via Whh·h₃, but vanilla RNNs struggle with long-range negation — this is exactly why LSTMs and GRUs were invented!
