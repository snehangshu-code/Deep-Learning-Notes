What actually happens

Forward Propagation — Plain English

🧠 The Core Idea

At each time step, the RNN cell does one job: combine what it's seeing right now (input xₜ) with what it remembers from before (hidden state h₍ₜ₋₁₎), and produce a new memory (hₜ). That's it.

THE TWO EQUATIONS — THIS IS ALL OF FORWARD PROP
hₜ = tanh( Wₓₕ · xₜ + Wₕₕ · h₍ₜ₋₁₎ + bₕ )
ŷₜ = softmax( Wₕᵧ · hₜ + bᵧ )
💡
Why tanh? The raw sum Wₓₕ·xₜ + Wₕₕ·h₍ₜ₋₁₎ can be any large number. tanh squishes it into the range [-1, +1]. This keeps the hidden state values controlled so they don't explode as we go through many time steps.
🔁
The Recurrent Part: Notice that hₜ feeds back in as h₍ₜ₋₁₎ at the next time step. This is the "recurrent" connection — the network's memory. Same weights Wₕₕ are used every single time step.

Why Does Each Part Exist?

Explore each component ↓
🔴 Why multiply xₜ by Wₓₕ?click to expand

Wₓₕ is a learnable weight that decides how important the current input is to the hidden state.

If Wₓₕ = 0 → input is completely ignored
If Wₓₕ = 1 → input passes through unchanged
If Wₓₕ = -2 → input is flipped and amplified

During training, backprop adjusts Wₓₕ so the network learns exactly how much to "listen" to each new input.

🔵 Why multiply h₍ₜ₋₁₎ by Wₕₕ?click to expand

Wₕₕ controls how much of the past memory is carried forward.

If Wₕₕ ≈ 0 → past is forgotten quickly
If |Wₕₕ| ≈ 1 → past is remembered strongly
If |Wₕₕ| > 1 → past is amplified (can explode!)

This is also the matrix that causes the vanishing gradient problem — multiplied many times across time steps.

🟡 What exactly does tanh do to the sum?click to expand

The raw sum Wₓₕ·xₜ + Wₕₕ·h₍ₜ₋₁₎ + bₕ can be any number — potentially very large or very negative. tanh squishes it:

tanh(x) = (eˣ − e⁻ˣ) / (eˣ + e⁻ˣ)
tanh(-∞) → -1
tanh(0) → 0
tanh(+∞) → +1
→ tanh = 0.000

🟢 What is the output ŷₜ used for?click to expand

Not all RNN tasks need an output at every step. It depends on the task:

Task TypeWhen Output UsedExample
Many-to-ManyEvery stepNamed entity tagging
Many-to-OneOnly last stepSentiment: "positive/negative"
One-to-ManyEvery step (gen.)Music / text generation
🎯
The big picture in one sentence: Forward propagation is just repeatedly asking "Given what I remember (h) and what I'm seeing now (x), what should I remember next?" — and the answer is always: hₜ = tanh(Wₓₕ·xₜ + Wₕₕ·h₍ₜ₋₁₎ + bₕ)
